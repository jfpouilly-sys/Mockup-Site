Accessibility Statement

AZIT is committed to making its website accessible in accordance with Article 47 of Law No. 2005-102 of February 11, 2005.

This accessibility statement applies to the website www.azit.com.

Compliance Status

The website www.azit.com is in partial compliance with the French General Accessibility Improvement Framework (RGAA) version 4.1.2, due to the non-conformities and exemptions listed below.

Test Results

The compliance audit conducted on January 2026 reveals that:

  • The compliance rate with RGAA version 4.1.2 criteria is being evaluated.
  • The average compliance rate of the online service is being improved.

Non-Accessible Content

The content listed below is not accessible for the following reasons:

Non-Conformities

We are actively working to address the following areas:

  • Some images may lack descriptive alternative text
  • Some interactive elements may not be fully keyboard accessible
  • Some color contrasts may not meet WCAG AA requirements

Exempted Content

No content is exempt from the accessibility obligation.

Disproportionate Burden Exemptions

No disproportionate burden exemption has been declared.

Establishment of This Accessibility Statement

This statement was established on January 2026.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (Vanilla)

User Agents and Assistive Technologies

Tests were conducted with the following combinations:

  • NVDA 2023 / Firefox / Windows 11
  • JAWS 2023 / Chrome / Windows 11
  • VoiceOver / Safari / macOS Sonoma

Tools Used to Evaluate Accessibility

  • WAVE (Web Accessibility Evaluation Tool)
  • axe DevTools
  • Contrast Checker
  • W3C HTML Validator

Feedback and Contact

If you are unable to access content or a service, you can contact the website manager to be directed to an accessible alternative or to obtain the content in another form.

  • Send a message: accessibility@azit.com
  • Contact AZIT: Contact Form

Remedies

This procedure is to be used in the following case: you have reported to the website manager an accessibility defect that prevents you from accessing content or one of the portal's services and you have not received a satisfactory response.

You can:

  • Write a message to the Defender of Rights
  • Contact the Defender of Rights delegate in your region
  • Send a letter by mail (free, no stamp required):
