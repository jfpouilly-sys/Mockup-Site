What is OPC UA FX?

OPC UA Field eXchange (FX) is the latest addition to the OPC UA family, designed specifically for deterministic field-level communication. It combines the flexibility and interoperability of OPC UA with the real-time performance required for industrial automation and control systems.

AZIT OPC UA FX Stack (In Development)

Our OPC UA FX implementation will provide a complete, production-ready solution for next-generation industrial communication. Leveraging our expertise in real-time protocols, it will feature tight integration with Time-Sensitive Networking (TSN) for deterministic data exchange.

Key Capabilities (Planned)

  • Deterministic real-time communication
  • TSN integration for time synchronization
  • OPC UA PubSub support
  • Field device profiles
  • Security and encryption
  • Seamless OPC UA integration
