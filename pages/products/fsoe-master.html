Product Overview

ISIT's Safety over EtherCAT protocol stack delivers certified functional safety without hardware constraints. As a software solution, it integrates seamlessly with your existing EtherCAT applications. MainInstance (Master) or SubInstance (Slave): reduce integration complexity, accelerate certification and maintain complete control over your hardware choices.

Key Benefits

  • IEC 61784-3-12 compliant: Fully standard and interoperable
  • Pre-certified up to SIL 3 (PLe): Reduces your certification time and cost
  • MainInstance / SubInstance versions: Implement full Safety solution
  • Easy API integration: Short learning curve for your teams
  • Interoperability: Works with most existing EtherCAT stacks
  • SiL Ready: Early safety network simulation powered by Acontis EC-Simulator
  • Expert support

White Channel Architecture

The white channel architecture means our FSoE MainInstance is independent of the underlying "black channel" EtherCAT master stack. This provides flexibility in choosing your EtherCAT implementation while maintaining full safety certification.

Quick Start Integration

The AZIT FSoE Master stack is designed for straightforward integration into safety controllers.

Initialization Example

C 
/* FSoE Master initialization */
#include "fsoe_master.h"

/* Configure master and connections */
FSOE_MASTER_CONFIG master_config = {
    .max_connections = 4,
    .cycle_time = 10  /* ms */
};

/* Initialize FSoE master */
if (fsoe_master_init(&master_config) == FSOE_OK) {
    /* Add slave connections */
    FSOE_CONNECTION_CONFIG conn = {
        .connection_id = 0x0001,
        .watchdog_time = 100,
        .safe_inputs = 8,
        .safe_outputs = 8
    };
    fsoe_master_add_connection(&conn);
}

Cyclic Processing

C 
/* Call in your cyclic safety task */
void safety_controller_task(void) {
    uint8_t safe_outputs[8];
    uint8_t safe_inputs[8];

    /* Read local safety inputs */
    read_local_safety_inputs(safe_outputs);

    /* Process all FSoE connections */
    fsoe_master_process();

    /* Get data from specific slave */
    fsoe_master_get_slave_data(0x0001, safe_inputs);

    /* Process safety logic */
    process_safety_logic(safe_inputs, safe_outputs);
}
