AZIT CANopen Master Stack

Our CANopen Master stack provides comprehensive network management and control capabilities for industrial controllers, PLCs, and gateway applications. Manage multiple CANopen slaves with full NMT, SDO, and PDO support.

Key Applications

  • Industrial PLCs and controllers
  • Motion control systems
  • Protocol gateways
  • HMI and SCADA systems
  • Test and diagnostic tools

Network Management

Full NMT master functionality for network configuration, slave monitoring, and error handling. Includes SDO client for parameter access and configuration of connected slaves.

Key Benefits

  • Multi-slave management

Quick Start Integration

Initialization Example

C 
/* CANopen Master initialization */
#include "canopen_master.h"

CO_MASTER_CONFIG config = {
    .bitrate = CO_BITRATE_250K,
    .sync_period = 10  /* ms */
};

/* Initialize CANopen master */
if (co_master_init(&config) == CO_OK) {
    /* Add slave nodes */
    co_master_add_node(10, "slave1.eds");
    co_master_add_node(11, "slave2.eds");

    /* Start network */
    co_master_start_network();
}

Network Management Example

C 
/* Network control */
void network_control(void) {
