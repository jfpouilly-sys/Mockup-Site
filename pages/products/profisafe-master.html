What is PROFIsafe?

PROFIsafe is the safety protocol extension for PROFINET networks, enabling SIL3/PLe safety communication in industrial automation. It's specified in IEC 61784-3 and provides a vendor-independent safety solution for real-time PROFINET systems.

AZIT PROFISAFE Master Stack

Our PROFIsafe Master implementation provides a complete, certified solution for building safety controllers and PLCs with PROFINET safety capabilities. Built with 30+ years of embedded systems expertise, it features a white channel architecture that separates safety logic from the underlying PROFINET communication.

White Channel Architecture

The white channel architecture means our PROFIsafe stack is independent of the underlying "black channel" PROFINET master stack. This provides:

  • Freedom to choose any PROFINET master stack (Siemens, Softing, etc.)
  • Simplified safety certification process
  • Clear separation of concerns in your codebase
  • Easier maintenance and updates

Quick Start Integration

The AZIT PROFIsafe Master stack is designed for easy integration with minimal API surface.

Initialization Example

C 
/* PROFIsafe Master initialization */
#include "profisafe_master.h"

PROFISAFE_SLAVE_CONFIG config = {
    .f_source_addr = 0x0001,
    .f_dest_addr = 0x0002,
    .watchdog_time = 100,  /* ms */
    .safe_inputs = 8,      /* bytes */
    .safe_outputs = 8      /* bytes */
};

/* Initialize PROFIsafe master */
if (profisafe_master_init(&config) == PROFISAFE_OK) {
    /* Start PROFIsafe communication */
    profisafe_master_start();
}

Cyclic Processing

C 
/* Call in your cyclic task */
void safety_task(void) {
    uint8_t safe_inputs[8];
    uint8_t safe_outputs[8];

    /* Read safe inputs from your application */
    read_safe_inputs(safe_inputs);

    /* Process PROFIsafe communication */
    profisafe_master_process(safe_inputs, safe_outputs);

    /* Write safe outputs to your application */
    write_safe_outputs(safe_outputs);
}

Integration Steps

  1. Add stack files to your project
    Include source files and headers in your build system
  2. Implement HAL functions
    Provide timestamp and black channel interface
  3. Configure PROFIsafe parameters
    Set F-addresses, watchdog, I/O sizes
  4. Integrate in cyclic task
    Call process function in your safety task
  5. Test and validate
    Use provided test cases and validation tools
