AZIT-SIM

Simulation de Réseau Industriel Temps Réel

Exécutez votre contrôleur industriel avec un réseau simulé. Développez et testez sans matériel physique.

Virtualisez les réseaux EtherCAT, CANopen et FSoE pour le développement, les tests et la mise en service.

Demande d'évaluation Télécharger la fiche technique
[Schéma de simulation réseau AZIT-SIM]

Virtualisez Votre Réseau Industriel

AZIT-SIM virtualise les réseaux industriels en simulant les équipements esclaves/subordonnés, vous permettant d'exécuter des applications maître sans matériel réel.

  • Simulez via adaptateur réseau physique ou virtuellement
  • Configurez avec vos fichiers ENI/EDS existants
  • Réagissez aux données de processus (PDOs)
  • Envoyez et recevez des données de service (SDOs)
  • Interagissez avec les machines d'état des équipements
  • Connectez-vous aux outils de jumeau numérique et de simulation

Accélérez Votre Flux de Développement

Développement Logiciel Maître

  • Exécutez l'application maître sans matériel
  • Testez les fonctions pour équipements indisponibles
  • Validez avec différentes configurations
  • Déboguez des topologies complexes

Tests Automatisés

  • Remplacez les tests manuels par des scripts
  • Simulez les erreurs d'équipement et changements d'état
  • Simulez les ruptures de câble et pertes de trame
  • Testez de grands réseaux

Mise en Service Virtuelle

  • Développez avec un jumeau numérique
  • Commencez les tests avant l'arrivée du matériel
  • Testez des scénarios dangereux en toute sécurité

Développement Firmware Équipement

  • Développez le firmware avant l'existence du matériel
  • Utilisez le débogage pratique de l'IDE
  • Testez avec des données enregistrées
  • Testez les transferts FoE, CoE, VoE

Modes de Simulation

Tous les Équipements Simulés

Le système sous test (SUT) communique via câble réseau physique avec AZIT-SIM fonctionnant sur du matériel externe. Aucun matériel spécial requis - un PC standard ou processeur embarqué avec Ethernet fonctionne.

Mélange d'Équipements Réels et Simulés

  • HiL avec ports multiples peut se connecter n'importe où dans le réseau
  • Mélangez équipements réels et simulés
  • Créez un jumeau numérique étape par étape
  • Simulez des erreurs à n'importe quel emplacement du réseau
[Schéma Matériel dans la Boucle]

Simulation Entièrement Virtuelle

Le réseau est simulé entièrement en logiciel sur la même machine que votre contrôleur. Aucun matériel séparé ni interface réseau physique requis !

AZIT-SIM remplace le pilote du contrôleur Ethernet, échangeant les trames directement avec votre application.

Avantages

  • Aucun matériel supplémentaire requis
  • Exécution dans VM ou environnement cloud
  • Parfait pour les pipelines CI/CD
  • Timing déterministe pour tests reproductibles

Protocoles Supportés

EtherCAT

  • Simulation réseau complète
  • Support horloges distribuées
  • Boîte aux lettres CoE, FoE, VoE, SoE
  • Jusqu'à 2000 équipements

CANopen

  • États NMT
  • PDO/SDO
  • EMCY
  • Dictionnaire d'objets

FSoE

  • Simulation de sécurité
  • États fail-safe
  • Injection d'erreurs

CANopen Safety

  • Simulation SRDO
  • Test d'état sûr

J1939

  • Simulation de messages
  • Messages DM
  • Réclamation d'adresse
