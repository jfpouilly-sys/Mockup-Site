What is J1939?

SAE J1939 is the standard communication protocol for heavy-duty vehicles including trucks, buses, construction equipment, and agricultural machinery. It defines network architecture, message formats, and diagnostic functions for CAN-based vehicle networks.

AZIT Stack J1939

Our J1939 implementation provides a complete, production-ready solution for vehicle ECU development. Built on decades of automotive expertise, it supports all major J1939 features including transport protocol, diagnostics, and proprietary PGN handling.

Quick Start Int√©gration

The AZIT J1939 stack is designed for easy integration with a simple, well-documented API.

Initialization Example

C 
/* J1939 stack initialization */
#include "j1939.h"

J1939_CONFIG config = {
    .name = {
        .identity_number = 12345,
        .manufacturer_code = 123,
        .ecu_instance = 0,
        .function_instance = 0,
        .function = 0,
        .vehicle_system = 0,
        .vehicle_system_instance = 0,
        .industry_group = 0
    },
    .preferred_address = 0x80
};

/* Initialize J1939 stack */
if (j1939_init(&config) == J1939_OK) {
    j1939_start();
}

Sending PGNs

C 
/* Send a J1939 PGN */
void send_engine_speed(uint16_t rpm) {
    J1939_MESSAGE msg;

    msg.pgn = 61444;  /* Electronic Engine Controller 1 */
    msg.priority = 3;
    msg.dest_addr = 0xFF;  /* Broadcast */
    msg.data_len = 8;

    /* Encode engine speed (0.125 rpm/bit) */
    uint16_t speed = rpm / 0.125;
    msg.data[3] = speed & 0xFF;
    msg.data[4] = (speed >> 8) & 0xFF;

    j1939_send(&msg);
}
